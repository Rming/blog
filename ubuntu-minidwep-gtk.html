Ubuntu下图形界面无线wep/wpa密码破解工具minidwep-gtk

3月 11 2011

之前用过bt3亲试，破解了邻宿舍的无线密码，觉得bt3已经很粗错了，没想到今天去朋友的博客一看，发现在ubuntu之下还有一个图形化的无线密码破解工具。下面是原文，只是作为参考，以备日后之用。

对于BT3 BT4系列的无线密码破译工具大家可能都有耳闻，今天奶牛为大家介绍的是一款ubuntu下图形界面的无线密码破解工具minidwep-gtk。支持中文，国人编写。奶牛也看到过cdlinux版本，可以用cdlinux制作启动U盘工作。

对于ubuntu下的minidwep-gtk，奶牛觉得方便很多，因为不再需要手动进行aircrack-ng的命令了。软件界面就4个按钮，扫描，启动，停止，还有退出。点击扫描会对无线热点进行扫描，选中要破解的对象，启动，就开始自动破解了，下面就是等待了。其它的不解释了，网上也有一些视频，大家可以看看。这个软件图形化，操作也很简单，奶牛不上图了。

这个软件的目的是检测无线密码安全，请勿用于非法用途。

最后说下安装，软件需要aircrack-ng的支持。

sudo apt-get install aircrack-ng

然后下载minidwep-gtk的安装包，奶牛收集了两个版本，都可以使用。如果程序菜单无法正常启动程序大家可以从终端中输入minidwep-gtk启动程序。
