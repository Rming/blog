线阵列式电容传感器远程测控系统设计

7月 1 2013

摘 要

  本课题研究设计了一种路基滑坡远程监测方法与装置，采用线阵列式电容传感器对滑坡断裂面位置进行实时检测，并通过以太网将测量数据传输到远程数据服务器进行存贮，通过任意的上网设备对远程数据服务器进行访问，实现对路基滑坡的远程实时监控。本装置基于单片机对线阵列式电容传感器的测量数据进行实时采集，利用网络控制芯片将单片机接入互联网，通过编制单片机和远程数据服务器之间的网络通信协议，实现了通过数据中心网站来将由单片机采集到的测量数据实时存储到远端服务器数据中心，并对远程设备进行控制。采用本次研究提出的解决方案，可以实现更加方便、快捷和高效的采集远程测量数据及控制被控对象，很好地解决了多地点测量及恶劣环境下的测量等难题。

  本装置工作流程如下：使用方形铝片制作的双极板电容线性排列在玻璃纤维片的弹性柔杆上，当弹性柔杆受到外部作用力产生弯曲形变时，带动电容片的位置变化，同时，电容极板间距变化，而上下结构对称的两个电容形成一对差动测量电容组，通过循环监测开关可以实现多组间的循环监测，然后将监测到的电容变化量通过A/D转换器传输到单片机，再由单片机通过GPRS网络传输到远程服务器上进行存储和处理，我们在使用浏览查看时只需要查询远程服务器上的数据即可进行本地的演示仿真。

  本系统的测量对象主要是柔杆的弯曲，这里只要电容传感器设计得当，这种测量方法是可以推广复制的，因此可以在其他弯曲测量的方面采用这种测量方法进行数据采集和监控测量，传统的GPS监测、光纤传感技术监测、激光测距设备监测、遥感监测等技术由于成本过高或者精确度不高而无法得到广泛应用，针对上述情况，本次设计研究的“线性阵列式电容传感器远程测控系统设计”，主要是针对目前各种传感器系统暴露的不足，设计的传感器结构更加简单、成本更低、可靠性更高，是更具市场竟争力的地质灾害监测预警系统。

关键词：远程控制，线阵列传感器，循环检测开关电路

Abstract

  This study is about the design of a remote monitoring method and apparatus for roadbed landslide. By using a linear array type of capacitance sensor, the position of the fracture surface of the roadbed landslide can be detected in real-time. The measurement data is then transferred to a remote server via Ethernet for data storage. Therefore, through any device which link to the Internet the remote server can be accessed, so as to realize the remote and real-time monitoring for roadbed landslide. The device which based on microcontroller measures the line array type capacitance sensor for real-time measurement data acquisition, By using the network control chip microcontroller connected to the Internet, Data center website store microcontroller real-time measurement data collected to data center server, and control the remote device through programming communication protocol between MCU and remote data servers.We can realize more convenient, fast and efficient distance measurement data and control the remote device by the solution proposed,which solving the problems such as location and bad environment measurement.

  This system work process is as follows: The square aluminum production double linear plate capacitor installation on the elastic soft stem of the fiberglass, Capacitance position changes when the elastic soft rod bending deformation by external forces, upper and lower structure symmetry form a differential measurement capacitance of the two groups,which can realize circulation between multiple sets of monitoring Through monitoring loop switch,then put the measured capacitance variation through the transmission of A/D converter to the microcontroller,SCM transfers data to the remote server for storage and processing through the GPRS network.We need to query the data on the remote server when processing the local demonstration simulation.

  As long as the capacitance sensor is designed properly, this method can promote replication, so it can be other bending measurement using this method for data acquisition and monitoring measurement.Traditional GPS monitoring, optical fiber sensing technology, laser ranging device monitoring, remote sensing monitoring technology, etc,which can not be used widely for high cost and low precision.We designs “the remote measurement and control system of Linear array of capacitive sensor” in view of the present all kinds of sensor system exposed shortcomings.It has Simpler structure, lower cost and higher reliability characteristics compared with the traditional detection methods,and will be the more market competitiveness of products. Key words: remote control, line array sensor, loop detection switch circuit.
