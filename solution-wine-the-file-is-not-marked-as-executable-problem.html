解决WINE “The file is not marked as executable”问题.

1月 14 2011

今天尝试了一下ubuntu下的wine1.2

sudo apt-get install wine1.2

安装很简单，按照人家的指示走就行啦，遵守协议，然后下载文件，然后安装完成啦。
本来想安装一个mini迅雷下载几个电影呢，悲剧的是下载下来了迷你迅雷的安装程序以后，在那个exe的文件（已经和wine建立关联）上双击，出现了

“The file is not marked as executable. If this was downloaded or copied form an untrusted source it may be dangerous to run.”

的提示，了解了一下才知道，需要在文件的属性上做一点修改才能解决。
解决办法：右击exe文件，属性–权限–允许以程序文件执行，在这个副选上打上勾就可以啦。

还算顺利。wine1.2+mini迅雷。偶尔会出现崩溃，真晕，不好用。

好用的版本请看这个：ubuntu迅雷wine-thunder_0.6-2_all.deb 完美运行
