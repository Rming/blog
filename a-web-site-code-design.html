常用功能模块分析

网站首页 (/)
文章阅读 post/

|->模块首页 ( 包含以下功能链接 ) ( user )
|->用户注册页 ( user/signup )
|->用户登陆页 ( user/login )
用户中心（user） |->用户信息页 ( user/profile ) 个人信息删改( edit ) 确认邮箱( confirm ) 更改邮箱( reconfirm ) 更改密码( edit ) 找回密码( resetpassword ) 更改头像( edit ) ( 或mod=? )
|->用户资源页 ( eg:文章管理、文件存储、相册管理，根据不同程序功能确定资源管理 )
|->文章管理 (user/post) 发表文章( new )、编辑文章( edit )、删除文章( del )、审核文章( checked ) (或mod=?)
|->相册管理 ( photo )
|->系统设置 ( option )

 

|–>管理员登陆 （admin/）
|–>用户管理 ( admin/user ) 增加用户( add ) 、审核用户（checked） 、删除用户 ( del ) 、编辑用户资料( edit )
后台管理（admin）|–>文章管理 ( admin/post ) 审核文章( checked ) 、 发表文章( new )、编辑文章( edit )、删除文章( del )、 (或mod=?)
|
|–>网站设置 (admin/set)
