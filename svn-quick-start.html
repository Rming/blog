SVN 与 Git 的区别

  • Git 是分布式的，SVN不是
  • Git 把内容按元数据方式存储，而SVN是按文件
  • Git 分支和SVN的分支不同（svn其实是文件夹）
  • Git 没有一个全局的版本号，而SVN有（Git 是 SHA1）
  • Git 的内容完整性要优于SVN
